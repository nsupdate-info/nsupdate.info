# French (fr) Translation Glossary

Agreed-upon translations for this project. These terms are **mandatory** — use them consistently in all French translations.

## Register

Use formal **"vous"** consistently throughout all translations. Do not use informal "tu/toi".

## Technical Terms (keep in English)

These terms should **not** be translated and must remain in English:

- API, DNS, DDNS, FQDN, HTTP, HTTPS, IPv4, IPv6, RFC, TLS, SSL
- nsupdate, ddclient, Fritz!Box, pfSense, OPNsense, Bind, bind9
- Pull-Request, Issue, Issue-Tracker
- nonce, token, cookie(s)
- dyndns2, HMAC_SHA512
- base64, LAN
