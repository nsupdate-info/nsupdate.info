# German (de) Translation Glossary

Agreed-upon translations for this project. These terms are **mandatory** — use them consistently in all German translations.

## Register

Use informal **"du"** (lowercase in running text, but **"Du/Dein/Dir/Dich"** capitalized as used in the existing translations) consistently throughout all translations. Do not use formal "Sie/Ihr/Ihnen".

## Technical Terms (keep in English)

These terms should **not** be translated and must remain in English:

- API, DNS, DDNS, FQDN, HTTP, HTTPS, IPv4, IPv6, RFC, TLS, SSL
- nsupdate, ddclient, Fritz!Box, pfSense, OPNsense, Bind, bind9
- Pull-Request, Issue, Issue-Tracker
- nonce, token, cookie(s)
- dyndns2, HMAC_SHA512
- base64, LAN
